 Well, if you want to create a website like this one, all you have to do is:
1. Install [Jekyll](https://jekyllrb.com/docs/installation/) on your machine.
1. Run these commands:
    * `jekyll new username.github.io`
    * `cd username.github.io`
    * `bundle exec jekyll serve`
1. go to `http://localhost:4000`'
1. And with that you have your **own website**!
1. If you edit something and want to rebuild your site, run `jekyll build username.github.io`
