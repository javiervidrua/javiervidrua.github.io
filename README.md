# javiervidrua.github.io
My webpage. I made it to learn jekyll and markdown, and to publish some things I find useful and interesting.

## Steps to create a website like mine
* `jekyll new username.github.io`
* `cd username.github.io`
* `bundle exec jekyll serve --livereload`
* `echo 'go to http://localhost:4000'`

## If you changed something and want to rebuild it
* `jekyll build username.github.io`
